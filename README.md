## üõ†Ô∏è Installation & Setup

Follow these steps to get MAOTH running locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/harshith046/MAOTH.git
   cd MAOTH
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```
